#  Todo List Application

## Описание проекта

Todo  - это десктопное приложение для управления задачами, разработанное на C# с использованием WPF . Приложение позволяет пользователям создавать, редактировать, отслеживать и удалять задачи с удобным интерфейсом.

##  Функциональность

### Основные возможности
-  **Создание задач** с названием, описанием, категорией и сроком выполнения
-  **Категоризация задач** (Дом, Работа, Учеба, Отдых)
-  **Отслеживание статуса** (активные/выполненные задачи)
-  **Управление задачами**:
  - Отметка задач как выполненных
  - Перемещение выполненных задач в историю
  - Удаление ненужных задач


##  Технологии

- **Язык программирования**: C#
- **Фреймворк**: .NET Framework / .NET Core (WPF)
